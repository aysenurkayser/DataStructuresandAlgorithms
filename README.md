# 2. Merge Sort Projesi

``[16,21,11,8,12,22]`` -> Merge Sort

* Yukarıdaki dizinin sort türüne göre aşamalarını yazınız.
* Big-O gösterimini yazınız.

```
[16, 21, 11, 8, 12, 22]

[16, 21, 11] || [8, 12, 22]

[16, 21] | [11] || [8,12] | [22]

[16] | [21] | [11] || [8] | [12] | [22] 

[16, 21] | [11] || [8] | [12, 22]

[11, 16, 21] || [8, 12, 22] 

[8, 11, 12, 16, 21, 22 ]


```

```
 Worst case   : O(n*logn)
 Average case : O(n*logn)
 Best case    : O(n*logn)
 ```
